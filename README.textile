h2. What is scribe-up ?

Scribe UP (for Scribe User Profile) is a web-oriented extension for the "Scribe":http://wiki.github.com/fernandezpablo85/scribe-java library. It retrieves the profile of a user after this user has been authenticated by an OAuth provider.
It's available under the Apache 2 license.

h2. OAuth providers supported

* Facebook

* Yahoo

* Google

* Twitter

* LinkedIn

* GitHub
